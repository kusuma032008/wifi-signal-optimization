# 3D WiFi Signal Distribution and Router Placement Optimization Using Triple Integration

## Overview
This project models WiFi signal distribution inside a classroom using multivariable calculus and triple integration.

## Features
- 3D WiFi signal simulation
- Dead zone detection
- Router placement optimization
- Signal visualization using contour plots

## Technologies Used
- Python
- NumPy
- Matplotlib
- Triple Integration
- Optimization

## Results
The project identifies the optimal router position that minimizes dead zone volume and improves signal coverage.
